# binaryTree
> **Not:** Sıralama üzerinde değişiklik yapılması gereken metot **incele** metodudur.

LNR (infix) için:
 ```ruby
 "#{sol.incele} #{dugum} #{sag.incele}"
```
LRN(postfix) için:
```ruby
"#{sol.incele} #{sag.incele} #{dugum}"
```

NLR(prefix) için: 

```ruby
   "#{dugum} #{sol.incele} #{sag.incele}"
```
